## 🐛 常见问题排查

### 问题1：配置添加失败
**检查**：
- [ ] API Key是否为空
- [ ] 所有必填字段是否填写
- [ ] 控制台是否有错误信息

### 问题2：编辑功能异常
**检查**：
- [ ] 点击编辑按钮是否进入编辑模式
- [ ] 字段值是否正确加载
- [ ] 保存/取消按钮是否正常工作

### 问题3：模型选择器不显示
**检查**：
- [ ] 是否有启用的配置
- [ ] 策略制定页面是否正确导入组件
- [ ] 控制台是否有JavaScript错误

### 问题4：连接测试失败
**这是正常的**：
- 测试使用的是模拟API Key
- 真实环境需要有效的API Key
- 主要测试UI交互是否正常
